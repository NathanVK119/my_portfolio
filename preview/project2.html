Summary

In summary, this QA project involved performing both manual and automated testing for the Poblacion Elementary School Portal System, focusing on the Student Registration and Data Management modules. It included creating and executing test cases, documenting bugs, preparing test data, and developing Cypress end-to-end scripts to validate core functionalities. The QA process ensured that the system worked reliably, reduced bugs, and improved both functionality and user interface before deployment. Although focused on testing, this project allowed me to gain valuable experience in QA practices, test planning, and automation, while strengthening my understanding of the software development lifecycle. This experience not only contributed to a higher-quality system for Poblacion Elementary School but also enhanced my skills in manual and automated testing, preparing me for future professional work in IT and software quality assurance.
